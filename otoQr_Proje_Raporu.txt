otoQr Projesi Detaylı Rapor
============================

Proje Adı: otoQr
----------------

Proje Amacı:
Araçlara özel QR kodları oluşturarak araç bilgilerini dijital ortamda yönetmeyi ve paylaşmayı amaçlayan bir web uygulamasıdır. Kullanıcılar araç bilgilerini sisteme ekleyebilir, yönetici paneli üzerinden araçları ve kullanıcıları kontrol edebilirler.

Klasör ve Dosya Yapısı:
-----------------------
- otoQr/: Ana web uygulaması (ASP.NET Core)
  - Pages/: Razor Pages (Admin, Login, Kullanıcı Paneli, Araç işlemleri)
  - Models/: Veri modelleri (Arac.cs)
  - Data/: Veritabanı işlemleri (AracVeritabani.cs)
  - wwwroot/: Statik dosyalar (CSS, JS, kütüphaneler)
  - appsettings.json: Genel yapılandırma dosyası
  - appsettings.Development.json: Geliştirme ortamı yapılandırması
- otoQr.AppHost/: Uygulama barındırma ve başlatma ile ilgili dosyalar
- otoQr.ServiceDefaults/: Ortak servis ayarları ve uzantılar

Kullanılan Teknolojiler:
-----------------------
- ASP.NET Core 8.0
- Razor Pages
- QRCoder (QR kod üretimi için)
- System.Text.Json (JSON işlemleri için)
- Bootstrap, jQuery (ön yüz için)
- SQL Server LocalDB (bağlantı dizesi hazır, ancak asıl veri json dosyasında tutuluyor)

Program.cs:
-----------
var builder = WebApplication.CreateBuilder(args);
// Add services to the container.
builder.Services.AddRazorPages();
var app = builder.Build();
// Configure the HTTP request pipeline.
if (!app.Environment.IsDevelopment())
{
    app.UseExceptionHandler("/Error");
    app.UseHsts();
}
app.UseHttpsRedirection();
app.UseStaticFiles();
app.UseRouting();
app.UseAuthorization();
app.MapRazorPages();
app.Run();

Data/AracVeritabani.cs:
-----------------------
public static class AracVeritabani
{
    private static readonly string DosyaYolu = "araclar.json";
    public static List<Arac> Araclar { get; set; } = new List<Arac>();
    static AracVeritabani()
    {
        YedektenYukle();
    }
    public static void YedektenYukle()
    {
        if (System.IO.File.Exists(DosyaYolu))
        {
            var json = System.IO.File.ReadAllText(DosyaYolu);
            var list = System.Text.Json.JsonSerializer.Deserialize<List<Arac>>(json);
            if (list != null)
                Araclar = list;
        }
    }
    public static void Yedekle()
    {
        var json = System.Text.Json.JsonSerializer.Serialize(Araclar, new System.Text.Json.JsonSerializerOptions { WriteIndented = true });
        System.IO.File.WriteAllText(DosyaYolu, json);
    }
}

        }
    }
}

Çalıştırma Talimatı:
--------------------
1. dotnet restore
2. dotnet run --project otoQr/otoQr.csproj

Notlar ve Uyarılar:
-------------------
- appsettings.Development.json dosyasının JSON formatı düzgün olmalıdır.
- Projede null referans uyarıları ve platform uyarıları (Windows’a özel API kullanımı) mevcut, bunlar gözden geçirilebilir.
- Kullanıcı şifreleri ve telefon gibi hassas bilgiler doğrudan saklanıyor, ek güvenlik önlemleri alınmalı.
- JSON dosyasının bozulmamasına dikkat edilmeli.
